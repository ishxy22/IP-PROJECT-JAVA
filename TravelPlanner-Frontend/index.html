About The Project

Our Mission

To redefine travel planning by creating a seamless, intelligent, and all-in-one digital dashboard. We empower travelers to move beyond simple spreadsheets and static documents, offering a dynamic platform that integrates AI-powered suggestions, real-time expense tracking, and personal memory logging to create truly unforgettable journeys.

Core Features

  • AI Itinerary Generation: Leverages the Groq API with the Llama 3.1 model to provide intelligent, context-aware travel suggestions, from landmarks to local cuisine.
  • Dynamic Timeline Planner: A visually intuitive timeline interface for adding and organizing daily activities, complete with timings and notes.
  • Travel Journal & Photo Gallery: Enables users to capture memories by adding daily notes and uploading photos directly to their itinerary, creating a rich travel log.
  • Expense Dashboard: An integrated module for budget management, featuring a real-time currency converter and analytics to visualize spending.
  • PDF Export: Allows users to download a beautifully formatted, offline-ready PDF of their complete itinerary for easy access on the go.

Technology Stack

Frontend

Built with Vanilla JavaScript (ES6+) for a lightweight and fast user experience, styled with Tailwind CSS for a modern, responsive design. The UI is enhanced with the AOS library for scroll animations.

Backend

Powered by a robust Java Spring Boot application, exposing a RESTful API for all data operations. It uses Spring Data JPA for seamless database interaction and is built with Maven.

Database

Utilizes an in-memory H2 Database for development and demonstration, configured for easy setup and rapid prototyping.

APIs & Services

Integrates the Groq API for AI features, and uses jsPDF & html2canvas for client-side PDF generation.
